[02] appstack-infra/ in progress

Microservices Platform Infrastructure

ECS on EC2RDS PostgresRedisOpenSearchSQSCloudFrontTerraform

Modular, free-tier-focused AWS infrastructure for a .NET microservices product platform (Angular admin + React Native app). A dedicated VPC with no NAT runs eight services plus PDF-ingest and search-sync workers on a single ECS-on-EC2 host behind an ALB. Two planes by design: a synchronous request path (ALB → ECS → Redis/OpenSearch/Postgres) and an async event path (S3 → SQS → containerized workers), each queue with a DLQ. Search hits OpenSearch only; the write path stays decoupled.

[03] cdc-infra/ in progress

Change Data Capture Pipeline

RDS PostgreSQLAmazon MSKDebeziumMSK ConnectKafkaTerraform

Terraform for a real-time CDC pipeline: every INSERT / UPDATE / DELETE on RDS PostgreSQL is streamed into Amazon MSK (Kafka) by a Debezium connector on MSK Connect — read straight from the write-ahead log, no polling. A dedicated VPC with a self-referencing security group lets RDS, MSK, and MSK Connect reach each other, and an S3 gateway endpoint pulls the Debezium plugin with no NAT Gateway. Logical replication is enabled via a custom RDS parameter group.
